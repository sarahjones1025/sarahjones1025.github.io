This counter application was done as a code challenge to stretch my javascript abilities.

The app takes an imput and counts up from zero in 1 sec intervals. You can reset the counter or restart it.

It also highlights if the number reached in count is divisible by 3, 5 or both!

I added some simple, original artwork and styled it to be playful and fun.

I completed this project from start to finish in 6 hours on a timer to really make it challenging!
